# ADR-0002: Sem IA no Escopo v1

## Status
Accepted

## Context
O foco atual e estabelecer base robusta para documentos, versoes, workflow, seguranca e auditoria.

## Decision
Nao implementar nenhuma funcionalidade de IA na v1.

## Consequences
- Positivas: menor risco de escopo, menor custo operacional inicial, entrega mais previsivel.
- Negativas: funcionalidades inteligentes ficam para fase futura via ADR novo.
